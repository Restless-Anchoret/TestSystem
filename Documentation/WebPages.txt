Веб-страница, видная гостю:

- Страница авторизации и регистрации.

Веб-страницы, видные участникам:

- Страница со списком всех соревнований (главная) (только видимые соревнования).
- Страница со списком всех тренировок (только видимые тренировки).
- Страница со списком задач по соревнованию.
- Страница с посылками по соревнованию (вместе с формой для отсылания новой посылки).
- Страница с монитором соревнования (для участников).
- Страница с кодом по посылке.
- Страница со статистикой участника.
	Список соревнований, в которых он участвовал и на которые зарегистрирован.
- Страница с регистрацией участника на модерируемое соревнование.
	Форма для ввода персональных данных.

Веб-страницы, видные модераторам:

- Страница для редактирования участников.
- Страница для добавления новых участников.
- Страница с актуальными результатами соревнования (для модераторов).
- Страница для регистрации участников на соревнование.
- Страница с результатами проверки посылки по запросу модератора.
- Страница со списком всех задач (глобальный архив задач).
	С кнопками "Редактировать задачу" у каждой задачи.
	С возможностью добавить новую задачу.
- Страница для редактирования задачи.
	С возможностью записывать новые группы тестов, новые авторские решения, изменять параметры задачи.
- Страница со списком всех соревнований и тренировок.
	С кнопками "Редактировать соревнование" у каждого соревнования. В списке есть как видимые, так и невидимые соревнования и тренировки.
	С возможностью добавить новое соревнование.
- Страница для редактирования соревнования.
	С возможностью добавлять новые задачи и изменять параметры соревнования.
	Кнопка "Начать системное тестирование".
	Кнопка "Завершить соревнование".
- Страница с результатами системного тестирования.

Веб-страницы, видные администраторам:

- Страница для редактирования модераторов.
- Страница для добавления модераторов.
